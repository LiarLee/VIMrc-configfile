## Installation guide
迁移和安装的说明： 
1. clone the project to temp foloder.
```bash
git clone git@github.com:LiarLee/VIMrc-configfile.git
```

2. move to .config
```bash
mv ./VIMrc-configfile ~/.config/nvim/
```

3. VIMPLUG Install or update.
```bash
PlugInstall
``` 

4. Coc install or update.
```bash
CocInstall
```

5. DONE.
